# Vyopar Project

Vyopar Project is a modern landing website for VYAPO, a smart POS management platform built for restaurants, retail stores, and hospitality businesses.

The website highlights POS billing, table management, mobile counter operations, revenue analytics, inventory workflows, client success stories, and contact details for demo requests.

## Features

- Responsive landing page built with Next.js and React
- Premium dark/light theme toggle with saved user preference
- VYAPO logo, favicon, and product preview assets
- POS laptop and mobile preview section
- Client success stories with real client logos
- Contact and footer section with phone, email, location, and social link placeholders
- Animated call-to-action buttons and polished card layouts

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React icons

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the site at:

```text
http://localhost:3000
```

## Available Scripts

```bash
npm run dev
npm run lint
npx tsc --noEmit
npm run build
```
